# Android Calculator (Python/Kivy) — Phase 1

Our project is a Python-based calculator application built with the Kivy framework for Android. 

## Features
-Numeric Entry 

-Decimal handling 

-Arithmetic Operations such as Addition, Subtraction, Multiplication, Division 

-Percentage Conversions 

-Error Handling 

-Expression Evaluation 

-Sign Toggle (±) (Negative/Positive Numeric Values) 

-Input Clearance (AC/C /Backspace buttons) 

# Tech Stack
- **Language:** Python 3
- **UI Framework:** Kivy
- **Platform:** Desktop (Phase 1 development/testing), Android packaging planned for later phases
